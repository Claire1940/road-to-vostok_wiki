## Mod設定メニュー (MCM) の導入

設定を調整するたびにゲームを再起動する手間を省くために、Mod設定メニューが必要です。このツールは既存のゲーム設定に直接統合され、インストールされたModのための使いやすいインターフェースを提供します。

| アクション | 説明 |
| :--- | :--- |
| **ダウンロード** | コミュニティのソースからMCMの.zipファイルを入手します。 |
| **配置** | .zipファイルを直接`/mods`フォルダにドラッグします。 |
| **解凍** | ファイルを**解凍しないでください**。ローダーは圧縮アーカイブとして読み取ります。 |
| **確認** | ゲームを起動し、設定メニューの右上に**MCMボタン**があるか確認します。 |

**Q: 設定にMCMボタンが表示されないのはなぜですか？**

A: これは通常、Steamの起動オプションでインジェクターパックが正しく設定されていないか、MCMの.zipファイルが解凍されている場合に発生します。ファイルが`/mods`フォルダ内で.zipのままであること、および起動パスが正しくインジェクターを指していることを確認してください。

**Q: 複数のModを同時に使用できますか？**

**Q: これらのModは製品版のリリースでも動作しますか？**
