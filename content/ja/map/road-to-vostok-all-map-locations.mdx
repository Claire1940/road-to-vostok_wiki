## ワールドマップとナビゲーションの理解

Road to Vostokにおいて、ナビゲーションは受動的なUI要素ではなく、能動的なゲームプレイメカニクスです。魔法のようなGPSは存在しません。代わりに、物理的なアイテムを見つけて利用し、自分の位置を把握する必要があります。タクティカルマップ（戦術地図）は、ワールドマップ全体を表示できる希少ながら不可欠なアイテムです。このゲームのマップはサバイバルジャンルの中でも最大級であるため、迷うことは生存に対する真の脅威となります。

世界はトランジットポイントで接続された様々なゾーンに分かれています。これらのトランジットはマップ間のゲートウェイとして機能し、特定の条件が必要な場合や、単に物理的な境界を見つけることで移動がトリガーされる場合があります。効率的に移動するためには、各ゾーンを定義するランドマークに慣れておく必要があります。

| マップ機能 | 説明 | 重要度 |
| :--- | :--- | :--- |
| **タクティカルマップ** | ワールド全体のレイアウトを示す回収可能なアイテム。 | **高** - 長期的な計画に不可欠。 |
| **トランジットポイント** | 次のエリアをロードする特定のゾーン（トンネルやゲートなど）。 | **最重要** - 進行または撤退の唯一の手段。 |
| **ランドマーク** | 特徴的な建物（学校、前哨基地、サイロ）。 | **中** - 地図なしで視覚的に方向を確認するために使用。 |
| **セーフゾーン** | 戦闘が制限されている、または商人が居住している限られたエリア。 | **高** - 回復や取引に必要。 |

> **警告:** トランジットポイントを明確に理解せずに新しいマップロケーションに入ると、医者や商人の元へ戻る簡単な方法がないまま、脅威度の高いゾーンに取り残される可能性があります。

## 学校（School）マップロケーション詳細ガイド

学校は、**Road to Vostokの全マップロケーション**の中でも最も象徴的で危険な場所の一つです。慎重な探索には報酬を与えますが、無謀な全力疾走には罰を与える多層的な環境です。地雷原のトランジット近くに位置する学校は、中ティアの物資回収と激しい近接戦闘の拠点となります。

### 学校への進入
学校に近づく際は、一般的に左側から回り込むのが安全です。このアプローチは自然な遮蔽物を提供し、遠距離光学機器を装備したプレイヤーがメインビル内の敵を偵察することを可能にします。このエリアには通常、少なくとも2〜3の敵スポーン地点があり、外周をパトロールしているか、学校に隣接する大きな家の中に潜んでいます。

### 主要な部屋と戦利品
学校内には、アクセスするために特定の鍵を必要とするロックされたエリアがいくつかあります。これらの鍵は必ずしも同じ場所にスポーンするとは限らないため、毎回のレイドが実用性を求める探索となります。

| 場所 | 必要な鍵 | 注目の戦利品 |
| :--- | :--- | :--- |
| **教室 (Classroom)** | 教室の鍵 | 医療品、包帯、文房具。 |
| **体育館 (Gymnasium)** | 体育館の鍵 | 大型クレート、潜在的な武器パーツ。 |

| **軍用クレート** | なし | 高ティアの弾薬と武器アタッチメント。 |

## マップロケーション間の戦術的移動

**Road to Vostokの全マップロケーション**間の移動には、弾薬と同じくらい自身の身体状態の管理が重要です。このゲームには、移動中の生存に直接影響を与える「武器スタンス（Weapon Stance）」メカニクスが備わっています。

### 武器スタンスと疲労
マウスホイールを使用して、高い武器スタンスと低い武器スタンスを切り替えることができます。
*   **ハイスタンス（High Stance）:** 武器を構えた状態。学校や前哨基地のような密集した場所で部屋に入ったり角を曲がったりする際に不可欠です。ただし、時間の経過とともに腕の疲労を消耗します。
*   **ロースタンス（Low Stance）:** 武器を下ろした状態。腕の疲労を回復させることができます。主要なランドマーク間の開けた野原を横断する際に使用してください。

### 夜間探索用の光源
報酬の多い場所の多くは暗闇に包まれていたり、深い内部構造を持っていたりします。これらのエリアを安全にナビゲートするには、光源を管理する必要があります。

1.  **Arva:** 基本的なポケット懐中電灯。最小限の光しか提供しませんが、完全な暗闇よりはましです。
2.  **Polaris:** 頭部装着型のライト。武器の操作のために両手を自由に保ちながら、優れた照明を提供します。
3.  **暗視ゴーグル (NVGs):** 高ティアの軍事拠点で発見されます。国境を越える夜間ナビゲーションの決定版です。

## 生存の必需品：クラフトと医療

どの**Road to Vostokの全マップロケーション**を訪れるにしても、インベントリには常に基本的な医療品を入れておく必要があります。このゲームの「ハードコア」な性質上、手足の骨折や重度の出血は数分でレイドを終わらせてしまいます。

### 主要なクラフトレシピ
スカベンジした材料を使用して、基本的な医療アイテムをクラフトできます。これは、マップの深部にいて医者のところまで戻れない場合に特に役立ちます。

| 完成アイテム | 材料 | 用途 |
| :--- | :--- | :--- |
| **副木 (Splint)** | 棒 + 布切れ + ダクトテープ | 骨折・ひびの修復。 |

| **包帯 (Bandage)** | 布切れ | 軽度の出血を止める。 |

> **ヒント:** 常にダクトテープを持ち歩きましょう。これは汎用性の高いクラフト部品であり、レイド中の修理や医療品のクラフトに頻繁に必要となります。

## 地形に応じた武器の取り扱い

Road to Vostokの武器は、複雑な相互作用を持つ物理的なオブジェクトです。滞在している特定のマップロケーションに合わせて武器を調整することは、重要なスキルです。

*   **スコープの調整:** レールに沿ってスコープを物理的に前後に動かすことができます。前方にセットされたスコープは学校のような近接戦闘の場所でのターゲット捕捉に適しており、後方にセットされたスコープは前哨基地の谷間を越える狙撃に適した視界を提供します。
*   **セレクター:** 常に武器を点検（デフォルト 'I'）して発射モードを確認してください。西側のライフルは通常左側にセレクターがありますが、東側のライフル（AKバリアント）はレバーを見るために点検ビューを右側に回転させる必要があります。
*   **キャンテッドスタンス:** レーザーアタッチメントがある場合は、キャンテッド（傾斜）スタンスを使用できます。これにより、遠距離用の光学機器を使いつつ、狭い通路ではレーザーを頼りに腰だめ撃ちの精度を高めることができます。

## FAQ

**Q: マップ上で自分の現在地を確認するにはどうすればいいですか？**

A: タクティカルマップに「現在地」マーカーは表示されません。学校、前哨基地などの主要なランドマークや、尾根や道路などの特定の地形的特徴を見て、自分の位置を判断する必要があります。**Road to Vostokの全マップロケーション**を把握するには、手動での偵察が必要です。

**Q: マップの移動は一方通行ですか？**

A: いいえ、トランジットポイントに到達できれば、マップ間を自由に行き来できます。ただし、ゾーンに再入場した際に敵がリスポーンしたり、配置が変わったりする場合があることに注意してください。

**Q: 高ティアの物資を見つけるのに最適な場所はどこですか？**

A: 学校や前哨基地内の軍事エリアや施錠された部屋が最高の報酬を提供します。オリーブドラブ色の軍用クレートを探してください。これらには最高の武器アタッチメントやNVGが入っていることが多いです。

**Q: 探索中に進行状況を保存するにはどうすればいいですか？**

A: 眠る場所を見つける必要があり、それには「毛布（blanket）」アイテムが必要です。一部のセーフゾーンや人里離れた家には毛布を使って休息できる場所があり、時間を進めたり現在の状態を保存したりする役割も果たします。
