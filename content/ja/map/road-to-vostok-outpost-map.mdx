## Outpostマップのレイアウトと主要な場所

Outpostマップは、SchoolやHighway（高速道路）から、より危険なMinefield（地雷原）エリアへと繋ぐ移行ゾーンです。Schoolの閉鎖的な廊下とは異なり、Outpostは広く開放的であり、異なる戦術的アプローチが求められます。プレイヤーは、中央の野原における長い視線と最小限の遮蔽物に対処しなければなりません。

| 場所 | 説明 | 主な特徴 |
| :--- | :--- | :--- |
| **スナイパータワー** | マップの端にある高い木造構造物。 | **高ティアの戦利品クレート**、9x19弾薬、食料。 |
| **ガンスミス施設** | 要塞化されたバンカー/トンネルシステム。 | **ガンスミスNPC**（10日目に帰還）、作業台。 |
| **倉庫 (The Warehouse)** | 馬の死骸が散乱する古い農場ビル。 | ナビゲーションの目印、時折床にアイテムが落ちている。 |
| **橋 (The Bridge) ** | 小さな川に架かる石造りの交差点。 | 危険なチョークポイント。しばしば山賊がパトロールしている。 |
| **軍事前哨基地** | 破壊された戦車や火砲が点在。 | **軍用クレート**、武器パーツ、医療品。 |

## 略奪戦略：最高の装備を見つける場所

**Road to Vostok Outpostマップ**での略奪には忍耐が必要です。都市部とは異なり、ここでの戦利品は軍用クレートの中に隠されていたり、放棄された車両の中に置かれていたりすることが多いです。

### スナイパータワー
境界近くに位置するこのタワーは、ゲーム序盤の物資を見つけるのに最も安全な場所の一つです。最上部では、9x19mm弾薬やツナ缶などの缶詰が入った軍用クレートを安定して見つけることができます。また、移動を開始する前に中央の野原に敵のパトロールがいないか偵察するのに最適な展望台としても機能します。

### ガンスミスのバンカー
ガンスミスはおそらくこのマップで最も重要なNPCです。しかし、彼は常にいるわけではありません。ゲーム内のイベントによると、ガンスミスは通常10日目あたりにOutpostに戻ってきます。彼の施設の近くには、進行状況を保存し、インベントリを管理できる二次シェルター（バンカー）があります。

### 猫の救出
Outpostマップにおけるユニークなインタラクションとして、バンカー/倉庫コンプレックス内で見つかる猫の「救出」ミッションがあります。猫を救出することは、ハードコアな生存生活の中での心温まる気晴らしになりますが、死なずに自宅のシェルターまで安全に戻る必要があります。フィールドで猫を失うと、そのランでは永久に失われてしまいます。

## 生存と戦闘の戦術

OutpostマップのAIは、最初の村よりも大幅に攻撃的です。ここの山賊はしばしばモシンやRK62を装備しており、遠距離からあなたを釘付けにすることができます。

### 狙撃手への対処法
銃声が聞こえて射手が見えない場合は、すぐに姿勢を低くしてください。現在のゲーム（2026年ビルド時点）には伏せ（プローン）のメカニズムはありませんが、破壊された戦車や乾草のロールの後ろにしゃがむことが唯一の防御策です。狙撃手は林縁で完全に静止していることが多く、わずか数ピクセルのようにしか見えません。

### 環境イベント：戦闘機
Outpostの最も没入感のある特徴の一つは「戦闘機」イベントです。約25%の確率で発生し、フィンランド軍のジェット機が低空飛行を行います。これらに殺傷能力はありませんが、エンジンの轟音は接近する山賊の足音をかき消してしまうため、屋外にいるのは危険な時間帯となります。

### 生存の必需品
| アイテムの種類 | 優先度 | 理由 |
| :--- | :--- | :--- |
| **水分補給** | **致命的** | Outpostには水源がほとんどありません。洋梨や桃の缶詰は不可欠です。 |
| **副木 (Splints)** | **高い** | 中央の野原での骨折は、移動ペナルティにより死刑宣告に等しいです。 |
| **光学機器 (Optics)** | **高い** | ここで一般的な長距離戦では、アイアンサイトでは不十分です。 |
| **止血帯 (Tourniquets)** | **中** | スナイパーのヒットによる重度の出血を止めるために不可欠です。 |

> **💡 ヒント:** 骨折しており水分も不足している場合、体力は急速に減少します。シェルターに到達できない場合は、最も価値のある戦利品を倉庫などの認識しやすいランドマークの床に置いてください。そうすれば、リスポーン後に回収できる可能性があります。

## 地雷原への移動

**Road to Vostok Outpostマップ**での探索が終わったら、次の目的地はおそらくMinefield（地雷原）でしょう。移動ポイントはマップの南東の角、農場施設の先にあります。地雷原では時間をかけて慎重に移動する必要があり、リソースを消耗するため、移動前に水分補給が50%以上であることを確認してください。

## FAQ

**Q: Outpostマップのどこにガンスミスがいますか？**

A: ガンスミスは、マップ中央北寄りにあるバンカー施設内にいます。彼はイベントベースのNPCであり、通常はゲーム内で10日目以降にのみ登場することに注意してください。

**Q: シェルターから遠い場所で骨折した場合、どうすれば生き残れますか？**

A: 副木がない場合、移動速度は大幅に低下します。最善の策は、痛み止めを服用して即座のHP減少を止め、最寄りのバンカーへ直行することです。死を避けられない場合は、装備が消えないように安全な隅にドロップしておきましょう。

**Q: Outpostの地図アイテムは見つかりますか？**

A: はい、地図アイテムは時折Generalistの店やSchool内のゴミ袋から見つかることがあります。ただし、それらは「ライブ」のGPS位置を表示するものではありません。スナイパータワーや倉庫などのランドマークを頼りにナビゲートする必要があります。

**Q: 戦闘機イベントの間はどうすべきですか？**

A: ジェット機の騒音を逆手に取りましょう。AIがあなたの移動音を「聞く」能力は周囲の騒音によって事実上無効化されるため、開けた場所を全力疾走したり、銃撃戦中にポジションを変更したりするのに最適なタイミングです。
