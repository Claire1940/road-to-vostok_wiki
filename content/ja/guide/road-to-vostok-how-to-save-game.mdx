## セーブと永続性の仕組みを理解する

Road to Vostokでは、セーブはポーズメニューにある手動操作ではありません。代わりに、ゲームは「オートセーブ」と「シェルター永続性」のロジックを利用しています。進行状況は主にシェルター（Area 05のキャビン）に紐付けられています。ストレージにアクセスしたり、ホームベースに入ったりすると、ゲームはインベントリとワールドの状態を記録します。

ワールド自体はダイナミックです。キャビンに入ってマップに戻るたびに、戦利品のスポーンとAIの配置がリフレッシュされます。これは、希少なインバーターや医療品などの特定のアイテムを効率的に集める「周回（ファーミング）」のための重要な戦術です。しかし、怪我、空腹、喉の渇きを含むキャラクターの身体状態は常に更新されています。野原の真ん中でゲームを終了すると、再開時に非常に不安定な状況に陥る可能性があります。

| 機能 | セーブの挙動 |
| :--- | :--- |
| **手動セーブ** | 利用不可。自動的な状態追跡を使用。 |
| **シェルターストレージ** | **永続的**に保存（パーマデスが発動しない限り）。 |
| **インベントリのセーブ** | マップの切り替え時、またはシェルターへの入室時に保存。 |
| **ワールドの状態** | シェルターへの出入り時にリセット/リフレッシュされる。 |

> **💡 ヒント:** ゲームを終了する前には必ずシェルターに戻り、ストレージを操作してください。これにより、最近獲得したすべてのアイテムが永続セーブファイルに登録されます。

## 3つのゾーン：リスクと報酬

セーブシステムはマップの地理と密接に関連しています。ゲームワールドは3つの異なる難易度階層に分かれており、それぞれ「失敗」した時のルールが異なります。これらのゾーンを理解することは、Road to Vostokのセーブ戦略を管理する上で極めて重要です。

### Area 05（スターターゾーン）
ここはあなたの安全地帯です。Area 05には主要なシェルターがあります。ここで死亡した場合、現在身につけている装備は失われますが、キャビンに保管されているものはすべて手つかずのまま残ります。これにより、備蓄を利用して悪い状況から立ち直る「リカバリーループ」が可能です。

### Border Zone（国境地帯）
国境地帯は、より強力なAI、地雷、ヘリコプターのパトロールが存在する移行エリアです。Area 05と同様に、ここでの死亡は手痛いものですが致命的ではありません。シェルターでリスポーンし、保管していたアイテムは無事ですが、持ち出したタクティカル装備はそのゾーンで失われます。

### Vostok（パーマデスゾーン）
ここからセーブの仕組みが完全に変わります。Vostokは、Mark 18や最高級のプレートキャリアなどの最高の戦利品が登場するエンドゲームエリアです。しかし、Vostokマップ内で死亡すると、ゲームは**セーブファイル全体を削除**します。装備を失うだけでなく、クエストの進行状況、シェルターのアップグレード、そしてスタッシュ（備蓄）のすべてを失うことになります。

| ゾーン名 | 死亡ペナルティ | セーブファイルの状態 |
| :--- | :--- | :--- |
| **Area 05** | 装備品の紛失 | **安全** |
| **Border Zone** | 装備品の紛失 | **安全** |
| **Vostok** | **全データ抹消** | **削除** |

## セーブの進行状況を守る方法

Vostokでは苦労して積み上げた進行状況が削除される可能性があるため、ベテランプレイヤーは「段階的」なセーブアプローチを取ります。国境地帯やVostokに足を踏み入れる前に、決して持ち出さない「ベース」レベルの装備をシェルターに確保しておくべきです。これにより、万が一セーブデータを失っても、次のランを素早く進めるための設計図を維持できます。

### クエストと減税
「保存」しておくべき最も重要なことの一つは、商人との評判です。トイレットペーパー、ローション、バケツなどのアイテムを探すタスクを完了することで、商人の税率を下げることができます。初期状態では税率が100%にも達することがあり、アイテムが法外に高価になります。クエストを完了するにつれてこの税金が下がり、やり直すことになった場合でも「新しい」セーブファイルの管理が非常に楽になります。

### リフレッシュループの活用
キャビンに入るとワールドがリフレッシュされるため、近くの家を効率的に漁ることで時間を「節約」できます。
1. シェルターに最も近い2つの家を漁る。
2. シェルターに戻り、アイテムを預ける。
3. シェルターを出て、戦利品のスポーンをリフレッシュさせる。
4. クエストに必要なコンポーネントが揃うまで繰り返す。

> **⚠️ 警告:** 国境地帯の地雷は固定配置ですが致命的です。高価値のクエストアイテムを運んでいる時は、決して急いではいけません。一歩のミスで、装備を失ったままシェルターに送り返されることになります。

## 装備とインベントリの管理

インベントリ管理は、セーブの二次的な形態です。体と装備を失う可能性があるため、アイテムを「必須」「取引」「クエスト」の階層に分類する必要があります。

| アイテムカテゴリ | アクション | 優先度 |
| :--- | :--- | :--- |
| **インバーター / バッテリー** | すぐにシェルターに保管する。 | **高** |
| **武器マガジン** | 2つを携帯し、予備は保管する。 | 中 |
| **食料 / 飲料** | 必要に応じて消費し、溜め込みすぎない。 | 低 |
| **医療品** | 常に包帯/薬を最低2つ携帯する。 | **高** |

本作はリアルな武器操作が特徴で、手動で弾薬を装填（ラッキング）し、マガジンを確認（デフォルトの「B」キー）する必要があります。チャンバーに弾を込めるのを忘れて死んだとしても、セーブシステムは容赦しません。シェルターエリアを離れる前に、常に武器がすぐに撃てる状態で「保存」されているか確認してください。

## Vostokランのための高度な戦略

ついにVostokゾーンに入ると決めた時、あなたはセーブファイル全体を危険にさらすことになります。これを軽減するために、準備こそが唯一の「セーブ」となります。

1. **夜間出撃:** 暗視装置（NVG）やフラッシュライトを使用してください。夜間はAIの索敵能力がわずかに低下するため、視線を切るチャンスが増えます。
2. **視線（LoS）:** AIに発見されると、見られている時間が長いほど彼らの精度が上がります。すぐに視線を切り、彼らの「エイム」クロックをリセットしてください。
3. **寝袋:** 使い捨てアイテムですが、寝袋を使えば危険な天候や暗闇をやり過ごすことができます。ただし、インベントリの大きなスペースを占有します。
4. **「ロードゲーム」による救済:** 現在のデモ版では、パーマデス以外のゾーンで死亡した場合、セーブを「ロード」することでシェルターに戻れることがあります。ただし、死亡時に身につけていた装備はすべて失われます。

## FAQ

**Q: Road to Vostokにはパーマデス（恒久的な死）の仕組みがありますか？**

A: はい、ただし「Vostok」ゾーンのみです。Area 05や国境地帯で死亡した場合は、携帯していた装備を失うだけです。Vostokで死亡すると、セーブファイル全体が削除されます。

**Q: ゲームを終了する前にどうやってセーブすればいいですか？**

A: 進行状況を確実に記録するには、シェルター（キャビン）に戻り、ストレージコンテナを操作してください。ゲームはマップの切り替え時やシェルターへの入室時に自動的に状態を保存します。

**Q: 国境地帯で死亡した場合、装備を取り戻すことはできますか？**

A: いいえ。一度死亡すると、そのキャラクターの遺体と持っていた装備はそのインスタンスから事実上消滅します。シェルターのストレージに以前「保存」しておいたアイテムを頼りに再装備する必要があります。

**Q: ワールドの戦利品をリフレッシュするにはどうすればいいですか？**

A: シェルターに入ってから出るたびに、戦利品とAIのスポーンがリフレッシュされます。これは、ゲーム序盤にインバーターや特定の武器マガジンなどのレアアイテムを効率的に集めるための最も効果的な方法です。

**Q: パーマデスを避けるためにセーブファイルをバックアップする方法はありますか？**

A: 技術的にはWindowsのAppDataフォルダからセーブフォルダを見つけることは可能ですが、このゲームは「ハードコア」な哲学に基づいて設計されています。セーブのバックアップは、Vostokゾーンの高リスクなゲームプレイという本来の意図に反するとみなされます。
