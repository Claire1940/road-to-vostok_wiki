国境地帯の過酷な環境を生き延びるには、鋭い観察眼と確かな射撃技術だけでは不十分です。拠点として呼べる「家」が必要になります。この**road to vostok furniture guide**は、冷たく何もない室内を、機能的で整理された作戦基地へと変えるために作られています。初期のCabinにこもる場合でも、離れた場所のテントに戦術的な前哨基地を設営する場合でも、環境をどう扱うかを理解することは長期生存の鍵です。この**road to vostok furniture guide**では、Decor Modeの細かな仕様、Furniture Catalogの仕組み、そしてアイテムグループ化の戦略的メリットを解説し、快適性と効率性を兼ね備えたシェルター作りを目指します。

## Decor Modeと配置の基本をマスターする

初期デモ版から2026年Early Access版への移行により、Road to Vostokにおける環境インタラクトは大幅に刷新されました。旧「Attic」シェルターでは基本的なアイテム配置のみ可能でしたが、現在のシステムでは家具を動的にフル操作できます。

空間のカスタマイズを始めるには、まず**Decor Mode**に入る必要があります。デフォルトでは、指定されたシェルター内で**F1**キーに割り当てられています。起動すると、重いワードローブから小さなスツールまで、あらゆる家具が操作可能なオブジェクトになります。

### 配置メカニクスとグリッドスナップ

オブジェクト移動時、ゲームは高度なシェーダーシステムで視覚フィードバックを提供します。緑の「valid」シェーダーは配置可能、赤の「invalid」シェーダーはめり込みや浮遊が発生することを示します。レイアウトをきれいに整理しやすくするため、システムには**grid snapping**が導入されています。これにより、壁際に木箱をきっちり並べたり、部屋の中央にベッドを無駄な隙間なく配置したりするのが簡単になります。

| Action | Control (Default) | Description |
|---|---|---|
| **Decor Modeに入る/終了する** | **F1** | カスタマイズUIとオブジェクトのハイライト表示を切り替えます。 |
| **オブジェクトを掴む/設置する** | **左マウスボタン** | ハイライトされた家具を持ち上げる、または配置します。 |
| **オブジェクトを回転する** | **R / マウスホイール** | 配置前に家具の向きを調整します。 |
| **距離を調整する** | **マウススクロール** | オブジェクトをプレイヤーに近づける/遠ざける操作です。 |
| **グリッドスナップ** | **自動** | オブジェクトを床面や近くの家具に揃え、隙間を防ぎます。 |

> **💡 ヒント:** 棚や絵画のような壁面要素には専用ロジックがあります。垂直面にカーソルを合わせたときのみ「valid」配置シェーダーが表示されます。

## Furniture Catalogシステム

2026年アップデートで最も革新的な機能のひとつが**Furniture Catalog**です。小さなシェルターを模様替えする際、床が大型オブジェクトで埋まっているとすぐにストレスになります。Catalogは家具用の一時的な「ポケット次元」として機能し、新しいレイアウトを決める前に部屋を完全に空にできます。

Decor Mode中は家具をCatalogへ「収納」できます。このUIはメインのプレイヤーインベントリと同様、**テトリス式インベントリ**を採用しています。ソファやベッドのような大型家具は大きなマスを占有し、スツールやナイトスタンドのような小型家具はより簡単に収められます。

### Catalogの利点:
1.  **模様替えが簡単:** アイテムを1つずつ動かすことなく、部屋全体を空けて新しい間取りを視覚化できます。
2.  **家具の移送:** Catalogに入れた家具は別シェルター間でも利用可能です。つまり、序盤Cabinのお気に入りベッドを、後半の上位シェルターへ移せます。
3.  **整理整頓:** 今は使わないが将来の前哨基地で使いたい家具を保管する、第二の収納として機能します。

## アイテムグループ化と収納ロジック

初期のサバイバルゲームでよくある不満は、「動かす前にコンテナを空にしなければならない」点です。Road to Vostokはこれを**Item Grouping**で解決しています。ランタン、工具、武器などのルート可能アイテムを家具の上に置くと、それらはその家具に「親子付け」されます。

Decor Modeで棚を持ち上げると、棚の上にあるアイテムは相対位置を保ったまま一緒に移動します。これにより、個々のルート品を細かく動かす手間をかけず、収納エリアを素早く再編成できます。

### 収納容量とコンテナ
家具システムはゲームのコンテナロジックと直接連動しています。ほとんどの大型家具は収納容器として機能します。ワードローブ、木箱、棚を複数配置することで、シェルターの収納容量を拡張し、何百ものユニークアイテムを保管可能です。医療品や弾薬を大量備蓄し、Vostokゾーン深部への長距離遠征に備えたいプレイヤーには不可欠です。

| Furniture Type | Primary Function | Interaction Type |
|---|---|---|
| **ベッド/スリーピングパッド** | セーブ地点 / 休息 | 固定サーフェス |
| **棚/テーブル** | アイテム展示 / 収納 | グループ化サーフェス |
| **ワードローブ/冷蔵庫** | 大容量収納 | 内部コンテナ |
| **カーペット** | 外観演出 | 重ね置き可能 |
| **ライトスイッチ** | 雰囲気/視認性 | インタラクティブ切替 |

## 新しい家具を入手する方法

初期シェルターにあるデフォルト家具だけに縛られる必要はありません。コレクションを増やす主な方法は3つあります。

### 1. 家具クラフト
シェルター内では**Crafting Menu**にアクセスできます。「Furniture」タブに各種レシピがあります。なお家具クラフトには**シェルター近接要件**があり、野外でベッドを作ることはできません。必ずホームベースの範囲内にいる必要があります。

| Craftable Item | Required Materials | Function |
|---|---|---|
| **ミリタリーベッド** | 枕x2、スリーピングパッドx2、毛布x1、寝袋x1 | 高ランクの休息地点。 |
| **シングルベッド** | 木材x1、毛布x1、マットレスロールx1、枕x1 | 標準的な休息地点。 |
| **ウォールシェルフ** | 木材x2、釘x1 | 基本的なアイテムグループ化用サーフェス。 |

### 2. 取引と物々交換
Road to Vostokのトレーダーは現在、家具用品も扱っています。汎用トレーダーはランダムで冷蔵庫やナイトスタンドを持っていることがありますが、「Grandma」のような専門トレーダーは、カーペット、布製品、居住快適品など特定カテゴリを扱う予定です。家具取引は高額になりがちで、さらに「梱包済み」状態の家具をシェルターまで運ぶために大きなインベントリ容量が必要になる場合があります。

### 3. スカベンジとルート
大型家具は稀ですが、小型の装飾品や一部の実用家具はワールド内で見つかります。高ティアのルートゾーンでは、希少な絵画や特殊コンテナを見逃さないようにしましょう。

> **Warning:** トレーダーから家具を運ぶ際は、帰路の安全を必ず確認してください。重い家具を担いだ状態は機動力が低く、敵対NPCにとって格好の標的になります。

## マルチシェルター戦略

Road to Vostokは**マルチシェルタープレイ**に対応しており、マップ上で複数拠点を同時に維持できます。これはゾーン深部へ進出したいプレイヤーにとって大きなゲームチェンジャーです。

-   **メインハブ:** 通常は初期Cabin。クラフトした家具の大半と長期保管物資を置く拠点です。
-   **バグアウトシェルター:** 高価値ルート地帯や脱出地点近くに置く、Tentのような小型拠点です。
-   **資源管理:** Furniture Catalogを使い、クラフトベンチや高ランクベッドなどの重要設備を拠点間で運搬し、どこでも安全に回復できる体制を作りましょう。

ゲームの開発状況やコミュニティについて詳しく知りたい場合は、最新の開発ログとパッチノートが掲載される[公式Road to Vostokサイト](https://roadtovostok.com/)をご覧ください。

## FAQ

**Q: 家具はワールド内のどこにでも設置できますか？**

A: いいえ。家具の設置とDecor Modeは指定されたシェルターゾーン内に限定されています。ただし通常のアイテム設置機能を使えば、小さなルートアイテムはどこにでも置けます。

**Q: 家具を削除した場合、中のアイテムはどうなりますか？**

A: 家具をCatalogへ移す場合、グループ化されたアイテムは基本的にそのまま紐づいた状態を保つか、家具が「梱包」される際に一時オーバーフローへ移されます。レシピ化に戻したり取引に出したりする前に、手動でコンテナを空にしておくほうが安全です。

**Q: 1つのシェルターに置ける家具数の上限はありますか？**

A: 厳密な「個数」上限はありませんが、実空間の広さとテトリス式Furniture Catalogが実質的な制限になります。シェルターを詰め込みすぎると、緊急時の移動が難しくなります。

**Q: このroad to vostok furniture guideにあるようなクラフトレシピは、どうやって増やせますか？**

A: レシピは、ワールドで設計図を見つける、特定トレーダーとの評判を上げる、または作業台付近で必要な基礎素材を所持していることで解放されます。

**Q: シェルター内の照明はカスタマイズできますか？**

A: はい。ほとんどの常設シェルターにはライトスイッチがあり、複数の光源を切り替えられます。これにより、雰囲気重視の「ムーディー」な演出にも、ルート整理用の最大視認性確保にも対応できます。
