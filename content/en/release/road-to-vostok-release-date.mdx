## Technical Performance and API Support

A major focus of the **road to vostok release date** was ensuring technical stability across a wide range of hardware. Because the game is built with a focus on realism and atmospheric lighting, the developer has provided multiple graphical API options. 

When launching the game via Steam, players are presented with several choices:

1.  **DirectX:** Recommended for most modern Windows-based systems.
2.  **Vulkan:** Often provides better performance on certain hardware configurations or Linux systems.
3.  **OpenGL:** A "bulletproof" backup API intended for players experiencing crashes or compatibility issues with the primary options.

> **Warning:** If you experience stuttering or crashes on launch day, try switching between DirectX and Vulkan in the Steam launch settings before adjusting in-game graphical presets.

The main menu now includes a hardware check tool and an in-game changelog toggle, allowing players to see exactly what has been updated in the 220+ additions and fixes included in the launch build.

Road to Vostok distinguishes itself from other survival titles through its deep commitment to player-driven content. The "Shelter" system has been expanded to support multiple locations.
