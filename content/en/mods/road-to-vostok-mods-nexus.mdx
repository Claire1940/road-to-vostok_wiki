<Steps>
  <Step num="1" title="Backup Your Game Files">
    Navigate to your *Road to Vostok* installation directory (usually `Steam/steamapps/common/Road to Vostok`). Create a copy of the entire game folder or at least the `Data` and `Config` folders. This ensures you can revert to a clean game state if a mod causes issues.
  </Step>
  <Step num="2" title="Download Required Modding Tools (If Any)">
    Some complex mods might require a specific "mod loader" or "API" that community members develop. Check Nexus Mods for any prerequisite tools that need to be installed before the actual mods. Follow their installation instructions precisely.
  </Step>
  <Step num="3" title="Download the Mod Files">
    Visit Nexus Mods or other reputable modding sites and download the desired *Road to Vostok* mod. Ensure you download the correct version for your game build (e.g., Early Access Build 1). Mods are typically packaged in `.zip` or `.rar` archives.
  </Step>
  <Step num="4" title="Extract and Place Mod Files">
    Extract the downloaded mod archive. The mod's instructions will usually specify where to place the extracted files. Common locations include the game's root directory, a `Mods` folder, or specific subfolders like `Data` or `Config`. Overwrite existing files only if the mod explicitly instructs you to do so.
  </Step>
  <Step num="5" title="Launch and Test the Game">
    Start *Road to Vostok* and check if the mod has been successfully applied. Pay attention to any new menus, visual changes, or gameplay alterations. If the game crashes or behaves unexpectedly, refer to your backup and try removing the mod.
  </Step>
</Steps>
