## Step-by-Step Installation Guide

Setting up the **road to vostok mod loader** requires a few manual adjustments to your Steam launch options. This ensures that the game loads through the custom injector rather than the standard executable.

### 1. Installing the Injector Pack
First, you must acquire the Injector Pack from the official community GitHub repository. This file serves as the engine for all subsequent modifications.

*   Locate your Road to Vostok installation folder (Right-click in Steam > Manage > Browse local files).
*   Download the Injector Pack and move the files directly into the main directory.
*   Open the **Properties** menu for Road to Vostok in Steam.
*   In the **Launch Options** field, paste the specific injector instruction provided in the pack documentation (typically pointing the executable to load the main pack via the injector).

### 2. Creating the Mods Folder
Once the injector is placed and the launch options are set, run the game once. The **road to vostok mod loader** will automatically detect the new environment and generate a `mods` folder within your directory. This is where all your future survival mods will reside.

## Implementing the Mod Configuration Menu (MCM)

To avoid having to restart the game every time you want to tweak a setting, you need the Mod Configuration Menu. This tool integrates directly into the existing game settings, providing a user-friendly interface for your installed mods.

| Action | Description |
| :--- | :--- |
| **Download** | Get the MCM .zip file from the community source. |
| **Placement** | Drag the .zip file directly into the `/mods` folder. |
| **Extraction** | **Do not extract** the files; the loader reads them as compressed archives. |
| **Verification** | Launch the game and look for the **MCM button** in the top right of the Settings menu. |

## FAQ

**Q: Does using the road to vostok mod loader get me banned from Steam?**

A: No. Road to Vostok is a single-player focused experience, and the developer (Antti) has been very supportive of the modding community. Modding the demo does not violate Steam's Terms of Service for this title.

**Q: Why isn't the MCM button showing up in my settings?**

A: This usually happens if the Injector Pack wasn't correctly set up in the Steam Launch Options or if the MCM .zip file was extracted. Ensure the file remains a .zip inside the `/mods` folder and that your launch path is correctly pointing to the injector.

**Q: Can I use multiple mods at the same time?**

A: Yes, the **road to vostok mod loader** is designed to handle multiple mod files. However, be aware of "Overhaul" mods that might change the same files, as they can occasionally conflict. Always check the mod description for compatibility notes.

**Q: Will these mods work on the full game release?**

A: It is unlikely. As the game moves from the demo to the full release in 2026, the underlying engine and code will change. Modders will need to update the **road to vostok mod loader** and individual mods to match the final version of the game.
