<Section icon={Shield} title="The Manual Backup Process">

Creating a **road to vostok backup save** is a straightforward process that should be performed before every gaming session or mod installation. Follow these steps to ensure your data is stored correctly.

<Steps>
  <Step num="1" title="Open the Run Command">
    Press the **Windows Key + R** on your keyboard to open the Run dialog box. Type **%appdata%** and hit Enter. This will take you directly to the Roaming folder.
  </Step>
  <Step num="2" title="Navigate to Road to Vostok">
    Locate the folder named **Road to Vostok**. Open it to reveal the internal save components, including the shader folders and configuration files.
  </Step>
  <Step num="3" title="Copy Core Data">
    Highlight all the **.TRES** files. While you can copy the entire folder, these specific files are the most important. Right-click and select **Copy**.
  </Step>
  <Step num="4" title="Create a Secure Storage">
    Create a new folder on your desktop or a cloud drive named **Vostok Backups**. Inside, create a subfolder with the current date (e.g., 2026-05-16) and paste the files there.
  </Step>
</Steps>

<Section icon={Sparkles} title="Safe Modding and Integration">

Modding in 2026 has become more streamlined with the Metro Mod Loader, but it still requires careful file management. If you plan to use mods like the Real Gun Names or the Mod Configuration Menu (MCM), your backup save is your safety net.

| Mod Component | Installation Path | Purpose |
| :--- | :--- | :--- |
| Metro Mod Loader | Root Game Folder | The core engine required to load external VMZ files. |
| Override Config | Root Game Folder | Allows the loader to prioritize modded assets over vanilla. |
| Mods Folder | Root/mods | A manually created folder where all .VMZ mods reside. |
| VMZ Files | Root/mods | Individual mod files containing textures, scripts, or data. |

To install mods properly, you must first locate your local game files via Steam. Right-click Road to Vostok in your library, select **Manage**, and then **Browse local files**. This is where you will place the mod loader and create your **mods** folder.

<Callout type="info" title="MCM Importance">
The **Mod Configuration Menu (MCM)** should always be the first mod you install. It provides the necessary user interface for adjusting settings within other mods, such as recoil sliders or attachment toggles.
</Callout>
