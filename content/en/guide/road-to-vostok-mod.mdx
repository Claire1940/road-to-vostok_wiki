## Step-by-Step Installation Guide

Setting up your first **road to vostok mod** involves modifying both the local AppData folders and the Steam installation directory. Follow these steps carefully to ensure a stable game environment.

### 1. Create a Backup
Before modifying any files, navigate to your local AppData folder. 
Path: `C:\Users\[YourUsername]\AppData\Roaming\RoadToVostok`
Create a backup of this entire folder. This contains your save games and existing settings, ensuring you can revert any changes if a mod causes a conflict.

### 2. Install the Metro Mod Loader
Download the Metro mod loader files from a trusted source like ModWorkshop. Extract the `modloadader.gd` file and place it directly into the game's main installation folder (where the `.exe` is located).

### 3. Configure the Steam Directory
Navigate to your game folder through Steam (Right-click > Manage > Browse local files). You must perform two critical actions here:
*   **Drop the `override.config` file:** This file is usually bundled with the mod loader and must sit in the root directory.
*   **Create a `mods` folder:** Simply right-click, create a new folder, and name it "mods" (all lowercase).

### 4. Adding Mods to the Folder
When downloading a **road to vostok mod**, you will often receive a `.zip` or `.vmz` file. 

> **Warning:** Do not extract the contents of individual mod archives into the mods folder. Place the zipped files directly into the `/mods/` directory as they are. The Metro Mod Loader is designed to read these archives without them being unpacked.

## Managing Settings via the MCM Menu

After launching the game with your mods installed, you will notice a "Mods" or "Updates" tab on the main screen. This indicates the loader is working. To fine-tune your experience, enter the **Settings** menu and look for the **MCM** tab.

Within the MCM, you can toggle specific features of your **road to vostok mod** suite:
1.  **Enemy Density:** If your frame rate is dropping, try lowering the density from "High" to "Medium."
2.  **Faction Relations:** You can toggle whether bandits fight each other or only target the player and guards.
3.  **Loot Rarity:** Increase the "Loot Multiplier" if you find the game too punishing, or lower it for a hardcore "zero-to-hero" run.

> **Tip:** Always check for updates within the Metro Mod Loader UI. Modders in 2026 frequently release patches to maintain compatibility with the latest game builds.

## Survival Tips for Modded Playthroughs

Modding the game can significantly increase the difficulty, especially with Faction Warfare active. Here are a few expert tips for surviving in a modded Vostok environment:

*   **Check Weapon Durability:** Many mods introduce stricter jamming mechanics. If you find a weapon with less than 50% durability, avoid using it in a firefight. It is likely to jam at the most critical moment.
*   **Carry a Splint:** With increased enemy density, fractures are much more common. Always keep a splint or an SOS kit in your quick-access slots.
*   **Listen Before You Leap:** Because AI factions now fight each other, you can often wait for a battle to conclude and then scavenge the remains. This "vulture" strategy is highly effective for gathering high-tier ammo and armor early on.
*   **Use the Highway with Caution:** The Highway map is notorious for snipers in the modded version. Ensure you have adequate cover and long-range optics before attempting to cross open areas.

## FAQ

**Q: Will using a road to vostok mod get me banned?**

A: No. Road to Vostok is a single-player focused experience, and the developer has historically been supportive of the modding community. However, always ensure you are downloading mods from reputable sites like ModWorkshop to avoid malware.

**Q: Why isn't my MCM menu appearing in the settings?**

A: This usually happens if the `override.config` file is missing from your main game directory or if the MCM zip file was extracted instead of being left as an archive in the `/mods/` folder. Re-verify your file structure against the installation table provided above.

**Q: Can I remove a road to vostok mod mid-playthrough?**

A: While some mods like Loot Modifiers can be removed safely, mods that affect your character's stats (like the XP Skill System) or save data may cause your game to crash if removed. Always keep a backup of your save folder before making changes.

**Q: What should I do if my game crashes on startup after installing mods?**

A: First, check the Metro Mod Loader UI for any update notifications. If the game still crashes, remove the most recently added mod from the `/mods/` folder one by one to identify the conflict. Ensure all files are placed in the correct directories as outlined in the 2026 installation guide.
